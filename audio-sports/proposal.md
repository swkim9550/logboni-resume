# 스포츠 오디오 중계 + 광고 모델 검토

> 작성일: 2026-03-26
> 아이디어: 야구 등 스포츠 중계를 "소리만 듣는" 저가/무료 이용권으로 제공하고, 광고로 수익화

---

## 1. 아이디어 요약

| 항목 | 내용 |
|---|---|
| 컨셉 | 스포츠 라이브 중계의 오디오만 스트리밍 (라디오 중계 느낌) |
| 타겟 | 영상까지는 필요 없는 라이트 유저, 출퇴근/운동 중 청취자 |
| 수익 모델 | 무료(광고 삽입) 또는 저가 이용권 (영상 이용권의 1/3~1/5) |
| 레퍼런스 | Spotify Free(광고), MLB Audio(유료 $3/월), 라디오 중계 |

---

## 2. 왜 이게 될 수 있는가

### 야구가 특히 잘 맞는 이유
- 야구는 원래 **라디오 중계 문화**가 강함 (한국: MBC/SBS 라디오, 미국: MLB Radio)
- 경기 시간이 3시간+ → 영상을 계속 보기 어려운 상황이 많음 (운전, 일하면서)
- 실황 중계 해설이 핵심 콘텐츠 — 화면 없이도 충분히 즐길 수 있음
- 축구/농구보다 **전개가 느려서** 오디오만으로 따라가기 쉬움

### 시장 상황
- 현재 스포츠 OTT는 전부 영상 기반 → 오디오 전용 상품이 없음 (블루오션)
- 기존 영상 이용권이 비싸다고 느끼는 유저층이 분명히 존재
- 스포티파이/팟캐스트로 스포츠 오디오 콘텐츠 소비가 이미 늘고 있음

### 유저 시나리오
```
출퇴근 지하철에서 이어폰으로 야구 중계 청취
→ 이닝 사이 15초 오디오 광고
→ 집 도착하면 영상으로 전환 (업셀)
```

---

## 3. 수익 모델 설계

### Option A: 무료 + 광고 (스포티파이 모델)

| 항목 | 설계 |
|---|---|
| 가격 | 무료 |
| 광고 | 이닝 사이, 투수 교체 시 오디오 광고 15~30초 |
| 제한 | 오디오만, 되감기 불가, 동시 1기기 |
| 업셀 | "영상으로 보기" 버튼 → 유료 이용권 전환 유도 |

광고 수익 추정:
- 야구 1경기 평균 9이닝 = 광고 슬롯 8~10회
- CPM(1000회 노출당) 오디오 광고: 약 $15~25 (한국 기준 2~3만원)
- 동시 청취자 1만명 기준: 경기당 약 20~30만원
- 월 30경기 × 20만원 = **월 600만원~** (초기)

### Option B: 저가 이용권 (월 1,900~2,900원)

| 항목 | 설계 |
|---|---|
| 가격 | 월 1,900원 또는 2,900원 |
| 광고 | 없음 (프리미엄 느낌) |
| 포함 | 오디오 라이브 + 하이라이트 클립(1분) |
| 업셀 | 영상 이용권 할인 쿠폰 제공 |

구독자 추정:
- 기존 영상 이용권 미구독자 중 5% 전환 가정
- MAU 50만 기준 → 2.5만명 × 2,900원 = **월 7,250만원**

### Option C: 하이브리드 (추천 ✅)

```
무료 티어: 오디오 + 광고 (이닝 사이)
라이트 티어 (1,900원/월): 오디오 + 광고 없음
기존 이용권: 영상 + 오디오 + 광고 없음
```

무료로 유저 풀 확보 → 광고 수익 + 유료 전환 동시 추구

## 4. 기술적 구현 난이도

### 쉬운 것들 (기존 인프라 활용)
- 오디오 스트림 추출: 기존 영상 스트림에서 **오디오 트랙만 분리** → FFmpeg 한 줄
- 스트리밍 인프라: 기존 CDN/HLS 그대로 사용 (오디오만이라 대역폭 1/50)
- 플레이어: 웹/앱에 오디오 플레이어 추가 (기존 영상 플레이어보다 훨씬 간단)

### 해야 할 것들
| 작업 | 난이도 | 설명 |
|---|---|---|
| 오디오 트랜스코딩 파이프라인 | 중 | 영상에서 오디오 분리 + AAC/MP3 인코딩 |
| 오디오 광고 삽입 (SSAI) | 중~상 | 서버사이드 광고 삽입, 이닝 사이 타이밍 감지 |
| 이용권/결제 체계 | 하 | 기존 이용권 시스템에 오디오 티어 추가 |
| 앱 UI | 중 | 오디오 전용 미니 플레이어, 백그라운드 재생 |
| 광고 SDK 연동 | 중 | 오디오 광고 네트워크 연동 (Google Ad Manager 등) |

### 대역폭 비용 절감 효과
```
영상 스트림: 3~5 Mbps (1080p)
오디오 스트림: 64~128 Kbps

→ 대역폭 약 1/40 수준
→ CDN 비용 거의 무시 가능
→ 동시 접속 10만명도 부담 없음
```

## 5. 현실적 리스크

### 🔴 중계권 이슈 (가장 큰 리스크)
- 현재 보유한 중계권이 **오디오 단독 배포를 허용하는지** 확인 필요
- 대부분의 스포츠 중계권은 "영상" 기준으로 계약 → 오디오만 별도 제공은 계약서에 명시 안 되어 있을 수 있음
- 리그/권리사와 별도 협의 또는 계약 수정 필요할 수 있음
- **KBO, NPB, MLB 각각 확인 필요**

### 🟡 광고 단가
- 오디오 광고 시장이 한국에서 아직 작음 (팟캐스트 광고 수준)
- 초기에는 광고주 확보가 어려울 수 있음 → 자사 광고(이용권 업셀)로 시작
- 스포츠 타겟팅이 가능하면 단가는 올라갈 수 있음 (맥주, 치킨, 스포츠용품)

### 🟡 기존 이용권 카니발라이제이션
- 영상 이용권 유저가 오디오로 다운그레이드할 위험
- 대응: 오디오 티어에는 다시보기/하이라이트 영상 미제공으로 차별화
- 실제로는 "안 쓰던 유저"가 오디오로 유입되는 효과가 더 클 가능성

### 🟢 긍정적 요소
- 신규 유저 유입 채널 (무료 → 유료 전환 퍼널)
- 서버 비용 거의 안 듦
- 경쟁사 없음 (국내 스포츠 오디오 스트리밍 전무)

## 6. 벤치마크

### MLB Audio (미국)
- 월 $3 (연 $30)
- 전 경기 오디오 라이브 + 아카이브
- MLB.tv 영상 구독($25/월)의 1/8 가격
- 꽤 오래전부터 운영 중 → 수요가 있다는 증거

### Spotify 모델
- 무료: 광고 삽입 (3~4곡마다 30초)
- 프리미엄: 광고 없음
- 전환율: 무료 → 유료 약 25~30%
- 스포츠에 적용하면 전환율은 더 높을 수 있음 (라이브 콘텐츠의 긴급성)

### TuneIn (미국)
- 무료로 스포츠 라디오 중계 제공
- 프리미엄($10/월)에 MLB/NFL 오디오 포함
- 광고 기반 무료 모델로 성장

## 7. 단계별 실행안

### Phase 1: MVP (1~2개월)
- KBO/NPB 야구 오디오 중계만 (가장 수요 높은 종목)
- 무료 + 이닝 사이 자사 광고 (이용권 업셀)
- 웹 전용 (앱은 나중에)
- 중계권 오디오 배포 가능 여부 법무 확인

### Phase 2: 확장 (3~4개월)
- 앱 백그라운드 재생 지원
- 외부 광고주 연동 (Google Ad Manager)
- 저가 이용권 출시 (1,900원/월)
- 축구/농구 등 타 종목 확대

### Phase 3: 고도화
- AI 실시간 자막/요약 (경기 상황 텍스트 푸시)
- 팟캐스트형 하이라이트 (경기 후 10분 요약)
- 카플레이/안드로이드 오토 연동

## 8. 결론

| 항목 | 평가 |
|---|---|
| 시장성 | ⭐⭐⭐⭐ — 야구 라디오 문화 + 라이트 유저 수요 확실 |
| 기술 난이도 | ⭐⭐ — 기존 인프라 활용, 오디오 분리만 추가 |
| 수익성 | ⭐⭐⭐ — 단독으로 큰 수익은 어렵지만 유저 유입 퍼널로 가치 있음 |
| 리스크 | ⭐⭐⭐ — 중계권 확인이 핵심. 이것만 해결되면 나머지는 수월 |
| 경쟁 우위 | ⭐⭐⭐⭐⭐ — 국내 경쟁자 없음 |

**현실성: 높음.** 기술적으로 어렵지 않고, 비용도 거의 안 들고, 경쟁자도 없음. 핵심은 중계권 계약 확인 하나. 이것만 통과하면 MVP는 빠르게 만들 수 있음. 야구 시즌(3~10월)에 맞춰 테스트해볼 만한 아이디어.
