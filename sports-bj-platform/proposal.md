# SPOTV NOW 스포츠 캐스터 플랫폼 — "NOW CAST"

> 작성일: 2026-04-27
> 상태: 아이디어 초안

## 브랜딩 네이밍

| 기존 용어 | NOW CAST 용어 | 컨셉 |
|-----------|---------------|-------|
| BJ | **캐스터 (Caster)** | 스포츠 캐스터 + 방송하다(broadcast)의 이중 의미 |
| 별풍선 | **골 (GOAL)** | 스포츠 = 골. "골을 넣다" = 후원하다. 직관적 |
| BJ 채널 | **캐스트 (Cast)** | "지금 캐스트 중" = 지금 방송 중 |

> 대안 후보: 골 외에도 **휘슬 (Whistle)**, **패스 (Pass)**, **킥 (Kick)** 등 검토 가능

## 1. 핵심 컨셉

**스포츠 전문 캐스터를 선발·육성하여, SPOTV NOW 안에서 라이브 스포츠 해설/토크 방송 + 서포터 후원 생태계를 구축한다.**

- 캐스터가 경기를 보며 실시간 해설·리액션·토크 진행
- **중계권 보유 경기**: 경기 영상과 함께 캐스터 해설 동시 송출 (편파 중계, 팬 중계)
- **중계권 미보유 경기**: 캐스터 음성/캠 + 실시간 스코어보드만으로 방송
- 서포터는 "골(GOAL)"로 캐스터에게 후원
- 캐스터는 서포터에게 **퀵뷰(1일 이용권)** 등 리워드 제공

## 2. 송출 구조 — 중계권 유무에 따른 분기

### 중계권 보유 경기 ✅

```
┌─────────────────────────────────────┐
│  경기 영상 (SPOTV 중계 화면)         │
│  ┌───────────────────────────────┐  │
│  │ 🎙️ 캐스터 오버레이 (PIP/캠)  │  │
│  │ 💬 실시간 채팅               │  │
│  │ ⚽ 골(GOAL) 후원 애니메이션   │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

- 기존 중계 영상 위에 캐스터 해설을 입히는 **듀얼 캐스트** 모드
- 공식 해설 vs 캐스터 해설 선택 가능 (시청자가 토글)
- 팀 편파 중계, 유머 중계, 전술 분석 중계 등 **다양한 관점** 제공
- → 같은 경기를 여러 캐스터가 각자 스타일로 중계

### 중계권 미보유 경기 ⚠️

```
┌─────────────────────────────────────┐
│  실시간 스코어보드 + 경기 타임라인   │
│  ┌───────────────────────────────┐  │
│  │ 🎙️ 캐스터 음성/캠 (메인)     │  │
│  │ 📊 실시간 스탯 데이터         │  │
│  │ 💬 실시간 채팅               │  │
│  │ ⚽ 골(GOAL) 후원              │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

- 경기 영상 없이 **캐스터의 해설·토크 자체가 콘텐츠**
- 실시간 스코어/스탯 데이터로 시각적 보조 (API 연동)
- 라디오 중계 느낌 — 캐스터 역량이 핵심
- → 중계권 없어도 서비스가 돌아가는 구조

```

[캐스터 구독]
```

## 4. 법적 검토

### ✅ 중계권 보유 경기
- SPOTV 중계 영상 위에 캐스터 해설 오버레이 → **자사 콘텐츠 활용이므로 문제 없음**
- 단, 중계권 계약에 "제3자 해설 삽입" 관련 조항 확인 필요

### ✅ 중계권 미보유 경기 (화면 미송출)
- 경기 화면/음성 송출 없이 캐스터 본인 해설·토크만 → **법적 문제 낮음**
- 스포츠 경기의 사실 정보(스코어, 기록)는 저작권 보호 대상 아님
- 아프리카TV에서 수년간 동일 모델 운영 중 (선례 존재)

### ⚠️ 주의 사항
- 경기 영상/중계 음성 직접 송출 → 중계권 침해 (절대 불가)
- SPOTV 명의 플랫폼이므로 개인 방송보다 리그 측 시선이 엄격할 수 있음
- **리그별 가이드라인 사전 확인 + 법무팀 검토 필수**

## 5. 캐스터 예측 랭킹 & 구독 시스템

### 핵심 구조

캐스터는 단순 해설자가 아니라 **경기 예측 전문가**이기도 하다. 예측 적중률이 곧 캐스터의 신뢰도이자 랭킹이 되고, 서포터는 이 랭킹을 보고 "누구의 캐스트를 들을지" 선택한다.

```
캐스터가 경기 전 예측 등록
  → 경기 결과에 따라 적중/실패 판정
    → 적중률 기반 랭킹 점수 변동
      → 랭킹 상위 캐스터 = 노출 우선 + 서포터 유입
        → 서포터가 신뢰하는 캐스터 구독
          → 구독자 많을수록 캐스터 수익 증가
```

### 예측 시스템

| 항목 | 내용 |
|------|------|
| 예측 대상 | 승패, 스코어, MVP, 첫 득점자, 이닝별 점수 등 |
| 등록 시점 | 경기 시작 전까지 (캐스트 시작 시 예측 공개) |
| 판정 | 경기 종료 후 자동 판정 (데이터 API 연동) |
| 난이도 가중치 | 단순 승패(1x) < 스코어 적중(3x) < 세부 예측(5x) |

### 캐스터 랭킹

```
[랭킹 점수 산정]
예측 적중률 (40%) + 서포터 수 (25%) + 골 후원액 (20%) + 시청 시간 (15%)

[등급 체계]
🥉 루키 캐스터     — 신규 (0~999점)
🥈 프로 캐스터     — 중급 (1,000~4,999점)
🥇 올스타 캐스터   — 상위 (5,000~9,999점)
🏆 레전드 캐스터   — 최상위 (10,000점+)
```

| 등급 | 혜택 |
|------|------|
| 루키 | 기본 캐스트 개설 |
| 프로 | 메인 페이지 노출, 퀵뷰 이용권 월 20장 지급 |
| 올스타 | 추천 캐스터 배너, 퀵뷰 월 50장, 골 수수료 30%로 인하 |
| 레전드 | 전용 프로모션, 퀵뷰 무제한, 골 수수료 25%, SPOTV 공식 콘텐츠 출연 기회 |

### 서포터가 캐스터를 선택하는 흐름

```
서포터가 "오늘 경기" 탭 진입
  → 해당 경기를 캐스트하는 캐스터 목록 표시
     ┌─────────────────────────────────────────┐
     │ 🏆 김프로  적중률 78% | 서포터 12.3K    │ ← 레전드
     │ 🥇 박축구  적중률 71% | 서포터 8.1K     │ ← 올스타
     │ 🥈 이야구  적중률 65% | 서포터 3.2K     │ ← 프로
     │ 🥉 최루키  적중률 --% | 서포터 0.1K     │ ← 루키
     └─────────────────────────────────────────┘
  → 서포터가 원하는 캐스터의 캐스트 입장
  → 캐스터의 해설 + 예측을 들으며 경기 관전
  → 마음에 들면 구독 / 골 후원
```

### 캐스터 구독

| 항목 | 내용 |
|------|------|
| 가격 | 월 3,900원 (캐스터별) |
| 혜택 | 전용 채팅 뱃지, 이모티콘, 캐스터 예측 알림, 구독자 전용 채팅방 |
| 수익 배분 | 캐스터 70% / SPOTV 30% |
| 효과 | 서포터는 신뢰하는 캐스터의 예측을 꾸준히 받아볼 수 있음 |

### 예측 랭킹이 만드는 선순환

```
캐스터가 예측을 잘 맞춤
  → 적중률 상승 → 랭킹 상승
    → 메인 노출 증가 → 서포터 유입
      → 구독·후원 증가 → 캐스터 수익 증가
        → 더 열심히 분석·예측 → 콘텐츠 품질 상승
          → 서포터 만족 → 구독 유지 (반복)
```

## 6. 캐스터 모집·운영

### 선발 (면접/오디션)

| 유형 | 특징 | 기대 효과 |
|------|------|-----------|
| 전문가형 | 전직 선수, 기자, 분석가 | 코어 팬 유입 |
| 엔터테이너형 | 입담, 리액션, 예능감 | 라이트 유저 유입 |
| 여성 캐스터 | 스포츠 관심 있는 여성 방송인 | 남성 팬 유입 + 여성 시청자 확대 |
| 인플루언서형 | 유튜브/아프리카 기존 팬덤 보유 | 즉시 트래픽 유입 |

### 운영
- 초기 **20~30명** 선발 (종목별 배치)
- 스포츠 지식 테스트 + 모의 방송 심사
- 방송 가이드라인 교육 (욕설·도박 조장·저작권 침해 금지)
- 성과 기반 등급제 → 상위 캐스터에게 프로모션 노출·퀵뷰 추가 지급

## 7. 골(GOAL) & 퀵뷰 이용권 구조

### 골 (GOAL) — 후원 화폐
```
1골 = 100원
최소 후원: 10골 (1,000원)
충전 단위: 50골 / 100골 / 500골 / 1,000골
결제: 카드, 인앱결제, 간편결제
수수료: SPOTV 35% / 캐스터 65%
```

### 후원 연출
- **골 후원 시**: ⚽ 골 애니메이션 + 서포터 닉네임 표시
- **100골 이상**: 🏟️ 스타디움 함성 효과
- **1,000골 이상**: 🏆 트로피 애니메이션 + 전체 채팅 공지

### 퀵뷰 이용권 (캐스터 → 서포터 리워드)
- 캐스터가 SPOTV로부터 이용권 수급 (성과 보상 or 할인 구매)
- 서포터에게 후원 감사 리워드로 제공
- **1일권 / 특정 경기 1회권** 단위
- 서포터: 골 후원 → 퀵뷰 수령 → 영상 시청 → 정기 구독 전환
