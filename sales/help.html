입력 도움말

하루 마감 입력 순서
1
매장과 날짜 확인

기본 매장은 세모두부입니다. 다른 매장을 입력할 때는 상단 매장 선택을 먼저 바꿉니다.

2
총 매출과 건수 입력

점심 매출/건수, 저녁 매출/건수를 입력하면 총 건수와 테이블 단가는 자동 계산됩니다.

3
판매 분류 입력

식사, 코스, 포장, 단품 매출을 입력합니다. 합계가 총 매출과 크게 다르면 마감 전에 다시 확인합니다.

4
메뉴와 주류·음료 입력

매장관리에서 등록한 항목이 매출 등록 화면에 표시됩니다. 없는 항목은 매장관리에서 먼저 추가합니다.

5
재료비와 경비 입력

재료비는 기본적으로 변동비입니다. 경비는 고정비와 변동비를 구분해서 입력합니다.

6
통장 금액 대조

통장입력 탭에서 보고 매출과 입금 금액을 대조합니다. 불일치 라벨이 보이면 차액을 확인합니다.

비용 구분 기준

고정비는 매출과 관계없이 일정하게 나가는 비용입니다. 변동비는 매출 또는 영업량에 따라 같이 움직이는 비용입니다.
변동비 예시
재료비, 주류, 식자재
고정비 예시
소모품, 수수료, 기타 경비

입력 전 확인

A
거래처명은 동일하게

같은 거래처는 같은 이름으로 입력해야 월별 거래처 비용 집계가 안정적으로 나옵니다.

B
빈 값은 0으로 처리

입력하지 않은 금액은 0원으로 저장됩니다. 실제 매출이 있다면 반드시 숫자를 입력합니다.

C
같은 날짜 재등록

이미 등록한 날짜를 다시 저장하면 기존 일자 데이터가 새 값으로 갱신됩니다.
