☁️ 삼성 SDS Cloud 방화벽 로그 서비스 개발

📅 2021.10 - 2022.04 👥 1인 주도 개발 🎯 백엔드 개발 / 시스템 설계

프로젝트 요약:

  • 삼성 SDS Cloud 환경의 VPC 및 Security Group 방화벽 로그를 저장·관리하는 서비스 개발
  • rsyslog를 기반으로 ELK 스택(Elasticsearch, Logstash, Kibana) 구성
  • Spring Batch를 활용해 로그 데이터의 주기적 백업 및 이력 관리
  • 방화벽 로그 데이터를 S3 Object Storage에 저장 및 자동 백업 서비스 구현

프로젝트 상세:

  • On-premise 환경에 직접 ELK 스택 설치 및 구성
  • rsyslog와 Logstash를 이용한 정확한 로그 패턴 수집 및 전송 파이프라인 구축
  • Spring Batch로 로그 데이터 백업 및 모니터링 이력 관리 시스템 개발
  • 백업 정책 수립을 통해 데이터 증가 시에도 안정적 시스템 운영 가능하도록 설계

역할 및 성과:

  • 설계부터 백엔드 개발까지 1인 주도 개발
  • On-premise ELK 및 Logstash 구성 경험 확보
  • 로그 처리 및 백업 정책 자동화를 통해 운영 안정성 향상
  • 사내 클라우드 로그 관리 서비스의 핵심 인프라 구축 기여

🆓 삼성 SDS Cloud Free Trial 서비스 개발

📅 2021.07 - 2021.10 👥 1인 개발 🎯 풀스택 개발 / 시스템 통합

프로젝트 요약:

  • 삼성 SDS Cloud의 신규 기능인 Free Trial(무료 체험판) 서비스 개발
  • 기존 클라우드 시스템에 MSA(Microservices Architecture) 방식으로 신규 서비스 통합
  • 무료 사용 신청을 위한 REST API 및 관리자 페이지 구현

프로젝트 상세:

  • 기존 운영 중인 클라우드 서비스의 구조(VPC, Security Group 등)를 학습 및 분석
  • 기존 Legacy 코드와의 연동 영향도를 최소화하도록 정밀한 통합 설계 수행
  • CRUD 기반의 Free Trial 관리용 REST API 개발
  • 신규 서비스 런칭 후에도 운영 중 서비스에 사이드 이펙트 없이 안정적 통합 성공

역할 및 성과:

  • 분석, 설계, 개발 등 전 과정 단독 수행 및 총괄
  • 서비스 런칭 이후 운영 안정성 확보
  • 성공적인 결과를 바탕으로 이후 주요 프로젝트에 추가 투입 기회 확보

사용 기술

Java Spring REST API MSA Kubernetes PostgreSQL ELK Stack
